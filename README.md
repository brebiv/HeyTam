# HeyTam website

Single Page Application website. Made with Django Framework, JQuery, Bootstrap.

## Installation

```bash
pip install django
git clone https://github.com/brebiv/HeyTam
cd HeyTam && python manage.py runserver
```
Now is on http://localhost:8000
